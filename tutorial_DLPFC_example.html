Tutorial for mNSF package¶

Yi Wang and Kasper Hansen¶

May, 2024¶

Abstract¶

mNSF is a package for spatial factorization for multi-sample spatial transcriptomics data

Spatial resolved transcriptomics opens the door to analyzing gene expression data in the context of spatial position. Recently, Townes and Engelhardt developed NSF, a non-negative spatial matrix factorization method. NSF can be used to identify spatial-dependent latent factors that are associated with functional anatomical regions. However, the model formulation is currently limited to a single sample.

However, an increasing number of spatial datasets contain multiple samples from different tissue sections. Properly modeled, such datasets could potentially allow us to identify matched regions across samples and increase the confidence in downstream analysis. However, there are few existing analysis methods for spatial data which allow joint analysis of multiple samples.

Here, we extend NSF to allow for multi-sample analysis. Our method, which we term mNSF, identifies factors which are common across multiple samples, and does not depend on cross-sample spot alignment, which is currently a challenging problem. In simulations, our method is capable of identifying spatial structures which are rotated between samples, as well as structures which only appear in some, but not all, samples. Using real data, we show our method is capable of identifying matched functional regions in multi-sample spatial transcriptomics data.

1. Introduction¶

The mNSF pacakge provides functionality to perform spatial factorization for multi-sample spatial data from spatial transcriptomics, or other multi-sample feature-by-observation high-dimensional matrix, where each observation has provided with the spatial location mNSF expects as input data: *feature-by-observation matrix, where each row is one feature(gene) and each column is one obsercation(spot) *the matrix for the coordinated of each observation correspondingly

mNSF expects as input parameter: *number of factors *number of samples

mNSF generate the output: *a factor matrix, where each column is one factor, for each sample *a loading matrix, where each row is correpsonding to one gene, and each column correpsonds to one factor
